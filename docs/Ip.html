IP Addresses, Packets and Routing

What's an IP address?

According to Wikipedia, an IP address stands for an Internet Protocol Address (IP Address). How does user and sender information get encoded and transmitted across IP Addresses? Good question, Wikipedia has some more suggested answers:
  1. Internet Protocol version 4 (IPv4) defines an IP address as a 32-bit number.
  2. However, because of the growth of the Internet and the depletion of available IPv4 addresses, a new version of IP (IPv6), using 128 bits for the IP address, was standardized in 1998
  3. The IP address space is managed globally by the Internet Assigned Numbers Authority (IANA), and by five regional Internet registries (RIRs) responsible in their designated territories for assignment to local Internet registries, such as Internet service providers (ISPs), and other end users.
